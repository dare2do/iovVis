# iovVis 前端子系统-报表生成系统
visualization subsystem - reporter.

## 快速入门 ##

一、Windows/Linux系统安装指南

1. 下载和安装nodejs（8.11.4），git工具（TortoiseGit）,克隆项目代码进入ui\reporter目录
2. npm install
3. npm run start

系统启动后，采用chrome浏览器，访问[http://localhost:30000](http://localhost:30000).
